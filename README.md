📖 Project Overview

This repository contains my Digital Literacy Portfolio, created as part of the CSE0001 Digital Literacy course at VIT Bhopal University.

As a Student Digital Ambassador, this project focuses on developing essential digital skills required for academic success and responsible online participation. The project includes activities such as creating digital awareness materials, building professional profiles, exploring online platforms, learning email etiquette, and understanding cybercrime prevention.

This repository is structured into five tasks, each representing one module of the Digital Literacy course. The purpose of this project is to help students build confidence in using digital tools effectively and safely.



This structure follows the required submission format specified in the project guidelines.
